* How to install evil mode manually 

   git clone --depth 1 https://github.com/emacs-evil/evil.git
   
   mkdir ~/.emacs.d/init.el
   
   (add-to-list 'load-path "~/Project/ITHelp/SampleCode/Emacs/evil")
   (require 'evil)
   (evil-mode 1)
